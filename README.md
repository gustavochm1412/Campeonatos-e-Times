
## 🔎 Acessando o Banco de Dados

O projeto utiliza banco H2 em memória. Para acessar o console:

1. Acesse `http://localhost:8080/h2-console`
2. Configure:
   - JDBC URL: `jdbc:h2:mem:campeonatodb`
   - Username: `sa`
   - Password: (deixe em branco)
